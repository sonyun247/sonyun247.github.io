# [운동일지](https://weight-log-swart.vercel.app/) <-

누구나 체계적인 운동을 할 수 있도록 도와주는 웹 어플리케이션 입니다.<br>
중구난방 그때 내키는대로 하는 운동이 아니라, 계획을 짜서 운동을 할 수 있습니다.<br>
오늘 한 운동의 종류와 무게, 횟수를 기록해두세요.<br>
