# [사워도우 베이커스 퍼센트](https://levain-calculator-kappa.vercel.app/) <-

사워도우 베이킹할 때 달라지는 원재료 배합량에 따른 비율 레시피를 제공해준다.<br>
베이킹 원재료 무게 변경에 따른 배합량을 매번 계산하는게 번거로워 제작하게 되었다.<br>
AI을 이용한 코딩을 해보고 싶어서 Gemini를 활용해봤다.<br>
